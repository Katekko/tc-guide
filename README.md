# Twilight Chronicle Flutter Web Spike

This is a non-destructive Flutter Web prototype for the guide site.

It mirrors the current Docusaurus home screen with:

- `go_router` route wiring
- Flutter generated localization files from ARB
- local image assets copied from the existing site
- responsive home sections for starter path, featured heroes, and guide sections

Run it locally:

```bash
flutter run -d chrome
```

Build the static web output:

```bash
flutter build web
```
